## Tech Stack

**Client:** Next.js, Apollo, CSS-in-JS with Emotion, Font Awesome 5

**Server:** Graphql api


  
## Features

- Persisting data on full page reload
- Server side rendering
- Progressive web apps
- Responsive

  
## Installation 

Install with npm

```bash 
  git clone https://github.com/ddsuhaimi/pokemon-next.git
  cd pokemon-next
  npm install
```

Run the app
```bash 
  npm run dev
```
Unit testing
```bash 
  npm run test
```
